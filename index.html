第一部分：python 基础支持

第二部分：python 应用实践

由于第一部分我们聚焦在 python 的基础知识的学习上，主要是让大家的属性 python 语法知识上，且能方便记录学习的过程，所以搭建环境我选择最简单的方式。 对于后面在项目实践的时候，会搭建适合项目开发的 python 开发环境。
